## 数组的转换方法

>## join()
>### 语法：
>#### arrayObject.join(separator)
>### 功能：
>#### 用于把数组中的所有元素放入一个字符串
>### 返回值：
>#### 字符串

>## reverse()
>### 语法：
>#### stringObject.reverse()
>### 功能：
>#### 用于颠倒数组中元素的顺序
>### 返回值：
>#### 数组

>## sort()
>### 语法：
>#### arrayObject.sort(sortby)
>### 功能：
>#### 用于对数组元素进行排序
>### 返回值：
>#### 数组
>### 说明：
>#### 1.即使数组中的每一项都是数组，sort()方法比较的也是字符串
>#### 2.sort()方法可以接收一个比较函数作为参数
